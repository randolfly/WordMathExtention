# 开发与调试（本机）

## 前置条件

- Node.js 18+（建议 20+）
- Windows 版 Word 桌面端（更适合调试 Office.js）
- 允许本机 https 开发证书

## 安装依赖

在仓库根目录运行：

- `npm install`

## 生成/安装本机开发证书

Office Add-in 在本机调试通常需要 `https://localhost`。

- `npm run dev-certs`

> 如果你已有证书方案，也可以自行配置 `vite.config.ts` 的 https 证书。

## 启动开发服务器

- `npm run dev`

默认地址：

- `https://localhost:3000/taskpane.html`

## Sideload 到 Word

此项目提供了一个便捷脚本（依赖 `office-addin-debugging`）：

- `npm run sideload`

如果你更偏好手动 sideload：

- 使用 `manifest/wordmath.xml`
- 将其加入 Word 的“受信任加载项目录”或开发者 sideload 流程

## 使用方式（MVP）

1. 在 Word 里点击 Home 选项卡的 `WordMath` 组按钮（`AsciiMath 公式`）
2. 在任务窗格输入 AsciiMath
3. 点击“插入/更新到 Word”
4. 若要编辑：在文档中选中已插入的公式控件，点击“从选中公式加载”，修改后再“插入/更新”

## 已知限制

- 仅识别由本插件插入的内容控件（`tag = wordmath`）
- AsciiMath 源码以隐藏元数据随公式自包含；若开启 Word“显示隐藏文字”，可能会看到标记文本
- KaTeX 预览与 Word 最终渲染可能存在细微差异（后续可统一渲染/字体策略）
